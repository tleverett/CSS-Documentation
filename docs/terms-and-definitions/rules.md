# Rules

For purposes of this documentation, a "Rule" or "Rule Set" is a [selection][selectors] followed by a [declaration block][declarations].

## Example

The following code example includes a single rule.

```text
.lorem .ipsum,                 -+
.dolor .sit {                   |
  amet: consectetur;            |
  adipiscing: elit !important;  |
}                              -+
```
